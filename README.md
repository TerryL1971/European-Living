### Helping the U.S. Military Community Explore Europe With Confidence

**European Living** is a modern travel, language, and cultural guide built specifically for Americans stationed in Germany.  
It removes the fear of exploring Europe by providing clear phrase translations, pronunciation help, curated local guides, and military-friendly recommendations.

The platform is built with **React + TypeScript + Vite**, styled with **Tailwind CSS**, and powered by **Supabase** for hosting, data, and storage.
